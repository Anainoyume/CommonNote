CSS 实现代码

/* 提示样式 - 蓝色 */
.quote-tip {
    background-color: #e8f4fd;
    border-left: 4px solid #3498db;
    padding: 15px 20px;
    margin: 15px 0;
    border-radius: 0 6px 6px 0;
}

.quote-tip::before {
    content: "✏️ 提示";
    font-weight: bold;
    color: #2980b9;
    display: block;
    margin-bottom: 8px;
}

/* 成功样式 - 绿色 */
.quote-success {
    background-color: #d4edda;
    border-left: 4px solid #28a745;
    padding: 15px 20px;
    margin: 15px 0;
    border-radius: 0 6px 6px 0;
}

.quote-success::before {
    content: "✅ 成功";
    font-weight: bold;
    color: #155724;
    display: block;
    margin-bottom: 8px;
}

/* 警告样式 - 橙色 */
.quote-warning {
    background-color: #fff3cd;
    border-left: 4px solid #ffc107;
    padding: 15px 20px;
    margin: 15px 0;
    border-radius: 0 6px 6px 0;
}

.quote-warning::before {
    content: "⚠️ 警告";
    font-weight: bold;
    color: #856404;
    display: block;
    margin-bottom: 8px;
}

HTML 使用方法

基本用法：

<div class="quote-tip">
    这是一个提示信息
</div>

<div class="quote-success">
    这是一个成功信息
</div>

<div class="quote-warning">
    这是一个警告信息
</div>

Markdown扩展语法（某些平台支持）

如果你的Markdown解析器支持自定义CSS类：

> [!TIP]
> 这是一个提示信息

> [!SUCCESS]
> 这是一个成功信息

> [!WARNING]
> 这是一个警告信息

注意：这种语法在GitHub、GitLab、Typora等平台有不同程度的支持。

在不同平台的使用建议

网站/博客：直接使用HTML + CSS的方式
GitHub：使用 > [!NOTE] 等语法
Typora：支持主题自定义引用样式
Notion：有内置的彩色引用块功能
你可以根据自己的需求调整颜色、图标和样式，创建更符合你项目风格的引用块。
